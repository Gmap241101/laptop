검증 제한
- package.json과 설치 의존성이 없어 실제 npm run build를 실행하지 못했습니다.
- Firebase CLI/Emulator가 없어 Firestore Rules 공식 컴파일을 수행하지 못했습니다.
- 브라우저 탭·창 종료 후 Firebase 지속성의 실제 동작은 테스트 브라우저와 실제 Firebase 프로젝트에서 확인해야 합니다.
- 동일 출처의 여러 탭 간 Firebase Auth 동기화 동작은 실제 브라우저 통합시험이 필요합니다.
